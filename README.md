


## Clone this project

```bash
> git clone https://github.com/setyawan12/setybot
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd setybot
> bash install.sh
```

### Usage
```bash
> npm start
```
